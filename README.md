# godo
Minimal task tracker written in Go.


## Installation

1. Install the version of Go for your system from `https://golang.org/doc/install`
2. Set the GOPATH using `export PATH=$PATH:$(go env GOPATH)/bin`
3. Clone the repository using `git clone https://github.com/AND2797/godo`
4. Run `sh godo/install.sh`
